This is a personal project simulating some gas.

Compiled in VSCode with CMakeTools.
To compile:
1. Create a 'build' directory
2. Configure CMake (in VSCode it's shift+ctrl+p and follow instructions)
3. Build project
4. 'Symulacja_Gazu.exe' executable should appear in 'build' directory

When you run a program, input number of particles to simulate.

To edit constants, visit "constants.h".
